# Installation de l'application android

1- Mettre le fichier "Contrôle Nao .apk" sur le téléphone.

2- Installer l'application en sélectionnant le ficher sur le téléphone

# INTERFACE DE CONTRÔLE ANDROID JAVA POUR NAO
David St-Onge, 05-2015

1-Installer l'environnement de développement Android Studio:
https://developer.android.com/sdk/index.html

2-Configurer le téléphone intelligent pour être en mode développeur et télécharger les pilotes:
http://developer.android.com/tools/device.html

3-Démarrer Android Studio puis ouvrer un nouveau projet via VCS/Github:
https://github.com/dastoqc/NaoAndroid

4- Ajouter une nouvelle configuration:
Éditer les configuration (menu déroulant près du bouton 'play'), ajouter une nouvelle application Android (+), sélectionner le module 'NaoAndroid_Conf' et laisser les autres options par défaut.

5-Brancher le téléphone Android par USB et lancer l'éxécution depuis Android Studio.

6-Connecter le téléphone au réseau Nao diffusé par le Nao lorqu'il est allumé.
Si le réseau n'apparait pas (attention le démarrage du Nao est très long!), brancher le Nao par cable réseau sur un ordinateur et télécharger le comportement 'tethering' par Choregraph. par le Nao lorqu'il est allumé.
