INTERFACE DE CONTRÔLE ANDROID JAVA POUR NAO
par David St-Onge, 05-2015

1-Installer l'environnement de développement Android Studio
https://developer.android.com/sdk/index.html

2-Configurer le téléphone intelligent pour être en mode développeur et télécharger les pilotes
http://developer.android.com/tools/device.html

3-Démarrer Android Studio et ouvrer le projet HelloAndroid sous 
jnaoqi-master\examples\android\HelloAndroid

4-Brancher le téléphone Android par USB et lancer l'éxécution depuis Android Studio.

5-Connecter le téléphone au réseau Nao diffusé par le Nao lorqu'il est allumé. Si le réseau n'apparait pas (attention le démarrage du Nao est très long!), brancher le Nao par cable réseau sur un ordinateur et télécharger le comportement 'tethering' par Choregraph.
